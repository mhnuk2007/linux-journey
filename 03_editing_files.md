# Section 3: Editing Files with nano and vim

## Topics Covered
- Opening and editing files
- Saving and exiting

## nano Cheat Sheet
```bash
Ctrl + O  # Write Out (Save)
Ctrl + X  # Exit
Ctrl + K  # Cut text
Ctrl + U  # Paste text
```

## vim Cheat Sheet
```bash
i       # Insert mode
Esc     # Exit insert mode
:w      # Save
:q      # Quit
:wq     # Save and quit
```

## Tip
Start with nano for beginners.
